# 今日市场扫描

## 你的任务
调用 `get_market_overview` 工具获取今日主要市场指数行情与用户持仓当日表现，给出一目了然的市场扫描。

## 分析步骤
1. **调用工具获取数据**
   - 调用 `get_market_overview`，拿到 6 大指数（标普500、纳指、道指、恒生、沪深300、上证）的实时行情
   - 拿到用户持仓的当日合计盈亏与 Top 涨跌个股
2. **指数概览**
   - 按市场分组（🇺🇸 美股 / 🇭🇰 港股 / 🇨🇳 A股）呈现各指数的现价、涨跌幅
   - 标出今日最强和最弱的指数
3. **持仓表现**
   - 引用工具返回的「持仓当日合计盈亏（USD）」
   - 列出当日贡献最大正盈亏 / 负盈亏的 Top 3 个股（来自 top_movers）
4. **一句话总结**
   - 综合指数与持仓，给出今日市场的整体定性（如"美股普涨，你的组合跟随上涨"）

## 输出格式
使用 Markdown：
- 指数用表格（市场 | 指数 | 现价 | 涨跌幅）
- 持仓涨跌用 `🟢` / `🔴` 标记方向
- 最后给一句加粗的总结

## 注意
- 行情数据来自实时工具调用，可能有轻微延迟，无需特别声明
- 如果某个指数返回为 null（抓取失败），如实标注"数据暂不可用"
- 不预测明日走势、不提供买卖建议
