# 季度回顾报告

## 你的任务
基于已注入的「当前投资组合快照」（包含近 1 年绩效指标、近期交易、当前持仓），帮用户生成一份**本季度**投资回顾报告。

## 报告结构
按以下章节输出 Markdown 报告：

### 1. 执行摘要
- 3–5 句话总结本季度的核心表现（收益、关键操作、关键事件）

### 2. 绩效表现
- 累计收益率、年化收益率、最大回撤、波动率、夏普比率
- 用一段话解读：这个成绩在合理区间吗？回撤是否可控？

### 3. 持仓结构变化
- 本季度新买入 / 清仓了哪些标的？
- 当前持仓的集中度与风格分布有何变化？

### 4. 关键操作复盘
- 挑出 2–3 笔本季度的关键交易，点评其得失

### 5. 下季度展望
- 基于当前持仓结构，指出 2–3 个下季度值得关注的重点

## 输出格式
- 严格按上述 5 个章节使用 `##` 二级标题
- 每节内容控制在 100–200 字
- 关键数字加粗，方便快速扫读

## 注意
- 报告面向用户本人，语气可以正式但不要官方套话
- 如果某项数据缺失（例如用户关闭了数据注入），明确指出"数据不足，建议开启「注入数据」后重试"
- 不提供具体买卖报价、不预测短期涨跌、不保证任何收益
