# 交易决策复盘

## 你的任务
基于已注入的「当前投资组合快照」中的**近期交易**（最近 20 条）和**当前持仓**，复盘用户近期买卖决策的合理性。

## 分析步骤
1. **交易概览**
   - 统计买入 / 卖出 / 分红（PAY）/ 开仓（OPEN）的笔数与金额
   - 标出金额最大的 3 笔交易
2. **决策合理性评估**（逐笔点评金额 Top 3）
   - **时机**：相对于当前价，买入价是高是低？卖出价是否兑现了收益？
   - **仓位**：这笔交易后，该标的的占比是否合理？
   - **成本**：均价是否被这笔交易显著抬高或摊薄？
3. **整体复盘**
   - 哪些决策事后看是正确的？哪些值得反思？
   - 是否存在"追涨杀跌"、"频繁交易"、"单笔过大"等模式？
4. **改进建议**
   - 给出 2–3 条针对未来交易的改进方向

## 输出格式
使用 Markdown：
- 用表格呈现 Top 3 交易的复盘
- 用"✅ 做对的"和"⚠️ 可改进的"两段呈现整体结论

## 注意
- 用现价（来自快照）事后评判，但要明确说明"这是事后视角"
- 不指责用户，保持客观、建设性
- 不提供具体买卖报价、不预测短期涨跌
