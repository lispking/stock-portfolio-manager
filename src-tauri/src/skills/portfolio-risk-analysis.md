# 持仓风险分析

## 你的任务
基于已注入的「当前投资组合快照」，对用户的组合做一次完整的风险体检，输出客观、可执行的风险评估。

## 分析步骤
1. **集中度分析**
   - 按市值计算每只持仓的占比，标出占比 ≥ 20% 的"重仓股"
   - 按市场（US / CN / HK）汇总，指出单一市场暴露是否过高
   - 按账户汇总，指出资金是否过度集中在一个账户
   - 按类别（现金类 / 分红股 / 成长股 / 套利）汇总，评估风格分布
2. **回撤与波动评估**
   - 引用快照中的「近 1 年最大回撤」「波动率」「夏普比率」
   - 给出主观评级：回撤 < 10% 低风险，10–20% 中等，> 20% 较高
3. **风险点清单**
   - 列出 3–5 个最值得关注的风险点（例如"单只股票占比 35%，集中度过高"）
   - 每个风险点给出一条**可执行**的缓解建议

## 输出格式
使用 Markdown：
- 先给一段 2–3 句的总体结论
- 然后用表格呈现集中度分布
- 最后用"⚠️ 风险点"列表呈现结论

## 注意
- 只基于快照中的真实数据，不要编造持仓或价格
- 区分「事实」（数据呈现的现象）与「判断」（你的推论）
- 不提供具体买卖报价、不预测短期涨跌
