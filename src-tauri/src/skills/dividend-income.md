# 分红收入分析

## 你的任务
调用 `get_dividend_income` 工具获取用户的分红/利息收入记录，分析被动收入的规模、来源与稳定性。

## 分析步骤
1. **获取分红数据**
   - 调用 `get_dividend_income`（days=365），拿到近 1 年的分红汇总
   - 工具返回：总净收入、笔数、按标的分解的净收入
2. **总览**
   - 近 1 年分红总收入（已扣除手续费）
   - 换算月均被动收入 = 总收入 / 12
3. **来源集中度**
   - 哪几只标的贡献了大部分分红？（Top 5）
   - 分红收入是否过度依赖单一标的？（Top 1 占比 > 50% 需提醒）
4. **分红频率**
   - 分红笔数反映出是季度派息还是年度派息？
   - 哪些月份分红到账最密集？
5. **被动收入评估**
   - 以当前组合总市值为基数，估算分红收益率（分红 / 总市值）
   - 与市场常见水平对比（美股分红股约 2–4%，REITs 约 4–8%）

## 输出格式
使用 Markdown：
- 总览用加粗的关键数字
- 按标的用表格（标的 | 净收入 | 笔数）
- 最后给一段"被动收入画像"的定性总结

## 注意
- 分红数据来自工具调用（PAY 类型交易），只统计有记录的部分
- 如果用户没有分红记录，明确告知"暂无分红记录"，不要编造
- 分红收益率是估算，标注假设（以快照总市值为基数）
- 不提供具体买卖报价、不预测短期涨跌、不保证任何收益
