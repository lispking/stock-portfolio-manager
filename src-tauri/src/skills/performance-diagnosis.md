# 绩效指标诊断

## 你的任务
基于已注入的「当前投资组合快照」中的**绩效指标**章节，为用户深度解读每一个指标的**含义、水平、改进方向**。

## 解读步骤
对以下每个指标，输出"含义 → 你的水平 → 评价 → 改进方向"四段式：

1. **累计收益率**
   - 含义：从期初到期末的总回报
   - 你的水平：（引用快照数据）
   - 评价：跑赢 / 持平 / 跑输常见基准（标普 500 长期约 8–10%/年）
   - 改进方向：……

2. **年化收益率**
   - 同上四段式，强调"年化"是把任意周期折算到年的口径

3. **最大回撤**
   - 同上四段式，重点解释回撤恢复的难度（回撤 50% 需要上涨 100% 才能回本）

4. **波动率**
   - 同上四段式，年化口径（日波动 × √252）

5. **夏普比率**
   - 同上四段式，参考标准：< 1 一般，1–2 良好，> 2 优秀

## 整体诊断
- 用 2–3 句话总结这份绩效"画像"最突出的特征（例如"高收益高波动，风险调整后收益一般"）
- 指出 1–2 个最值得改进的指标

## 输出格式
- 每个指标用 `### 指标名` 作为三级标题
- 四段式内容用紧凑的段落或 bullet，不要套话

## 注意
- 严格基于快照数据，不要编造收益率数字
- 评价基准要给出来源假设（例如"以标普 500 长期年化 ~9% 为参照"）
- 不提供具体买卖报价、不预测短期涨跌、不保证任何收益
